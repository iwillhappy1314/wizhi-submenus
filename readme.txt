== Installation ==

= For English user =

1. Upload `wizhi-submenus` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go go Appearance > Widget, put "Wizhi Submenus" in widget area

= 中文用户 =

1. 上传插件到`/wp-content/plugins/` 目录
2. 在插件管理菜单激活插件
3. 转到 外观 > 小工具, 把 "Wizhi Submenus" 拖放到需要显示子菜单的侧边栏即可
