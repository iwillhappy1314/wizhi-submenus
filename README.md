## 插件功能

在侧边栏显示子菜单，子菜单有两种来源。

1. 当前页面的子页面，或隶属于同一个父级页面的子页面。
2. 关联到某个文章类型的自定义分类法项目

联系方式：470266798@qq.com

## 安装方法

1. 上传插件到`/wp-content/plugins/` 目录
2. 在插件管理菜单激活插件
3. 转到 外观 > 小工具, 把 "Wizhi Submenus" 拖放到需要显示子菜单的侧边栏即可
